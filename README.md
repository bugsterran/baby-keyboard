# 🎹 아기 키보드 놀이터

> "아무 키나 눌러도 재밌는 일이 일어나요!"

영유아가 컴퓨터 키보드를 마음껏 두들길 수 있는 웹 놀이터입니다.
어떤 키를 눌러도 페이지가 꺼지지 않고, 화면에 알록달록한 시각 효과만 나타납니다.

**[배포 URL 추가 예정]**

---

## 핵심 기능

- **키 입력 = 시각 효과** — 모든 키보드 입력에 이모지 + 파티클이 터짐
- **시스템 단축키 차단** — ESC, Ctrl+W, Alt+F4, F5 등 전부 차단
- **소리 없음** — 시각적 효과만. 부모가 시끄럽지 않게
- **풀스크린 모드** — PWA 전체화면으로 동작
- **나가기** — 오른쪽 위 ✕ 버튼 클릭

## 키 입력 반응

| 키 | 효과 |
|----|------|
| A~Z | 테마별 이모지 + 한글 이름 팝업 + 파티클 |
| 0~9 | 테마별 이모지 + 파티클 |
| 스페이스바 | 화면 중앙 대형 폭죽 + 이모지 버스트 |
| 엔터 | 전체 화면 컬러 플래시 + 파티클 비 |
| 기타 키 | 랜덤 이모지 + 소규모 파티클 |

## 기술 스택

- **Vite** — 빌드 도구
- **Vanilla JS** — 프레임워크 없음
- **Canvas API** — 파티클 시스템 (오브젝트 풀링, 중력/마찰 물리)
- **CSS Animations** — 텍스트 팝업, 이모지 버스트, 화면 플래시
- **PWA** — manifest.json, 풀스크린 모드

## 프로젝트 구조

```
baby-keyboard/
├── index.html                  # 앱 진입점
├── public/
│   └── manifest.json           # PWA 매니페스트
├── src/
│   ├── main.js                 # 화면 라우터 (홈 ↔ 놀이)
│   ├── style.css               # 글로벌 스타일 + 애니메이션
│   ├── screens/
│   │   ├── home.js             # 테마 선택 화면
│   │   └── play.js             # 놀이 화면 (키 입력 → 효과)
│   ├── systems/
│   │   ├── keys.js             # 키 캡처 + 시스템 단축키 차단
│   │   ├── particles.js        # Canvas 파티클 엔진
│   │   ├── effects.js          # DOM 텍스트/이모지 팝 효과
│   │   └── fullscreen.js       # 전체화면 API 래퍼
│   └── data/
│       └── themes.js           # 12개 테마 정의 데이터
```

## 실행

```bash
npm install
npm run dev       # 개발 서버 (http://localhost:5173)
npm run build     # 프로덕션 빌드 (dist/)
npm run preview   # 빌드 결과 미리보기
```
