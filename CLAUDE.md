# 아기 키보드 놀이터 (Baby Keyboard)

> "아무 키나 눌러도 재밌는 일이 일어나요!"

## 컨셉
영유아가 컴퓨터 키보드를 마음껏 두들길 수 있는 웹 놀이터.
어떤 키를 눌러도 페이지가 꺼지지 않고 화면에 시각적 효과만 나타남.
부모는 아이가 키보드 치는 동안 자유 시간 확보.

## 핵심 요구사항

### 절대 규칙
1. **어떤 키를 눌러도 페이지가 꺼지지 않음**
   - ESC, Alt+F4, Ctrl+W, Ctrl+Q, F5 등 모든 시스템 단축키 차단
   - 뒤로가기 차단
   - 탭 전환 차단 (가능한 범위에서)
   - 우클릭 차단
2. **소리 없음** — 시각적 효과만. 부모가 시끄럽지 않게.
3. **풀스크린 모드** — PWA 전체화면으로 동작
4. **부모 종료 기능** — 특수 조합키로만 종료 (예: 5초간 Ctrl+Shift+Q 유지)

### 키 입력 반응
모든 키보드 입력에 화면에 비주얼 효과 발생:
- 글자 키 (A~Z): 해당 글자가 크게 나타남 + 테마별 이미지
- 숫자 키 (0~9): 숫자 + 효과
- 스페이스바: 큰 효과 (폭죽, 무지개 등)
- 엔터: 화면 전체 효과
- 기타 키: 랜덤 효과
- 동시에 여러 키: 여러 효과 동시 발생

### 비주얼 효과
- 폭죽 파티클 💥
- 글자/이미지가 크게 뿅 나타남
- 별똥별 쏟아짐 ⭐
- 무지개 색 퍼짐 🌈
- 풍선 올라감 🎈
- 하트 흩날림 ❤️
- 반짝반짝 스파클 ✨
- 색깔 물감 퍼짐

## 테마

| 테마 | 키 입력 시 | 배경 |
|------|----------|------|
| 🦁 동물원 | 동물 이미지 + 이름 | 초원/숲 |
| 🚗 탈것 | 자동차/비행기/배 | 도로/하늘 |
| 🌊 바다 | 물고기/해양생물 | 바닷속 |
| 🍎 과일 | 과일 이미지 + 이름 | 밝은 배경 |
| 🌈 색깔 | 화면 색 변경 + 도형 | 흰 배경 |
| 🎈 파티 | 폭죽/풍선/별 | 어두운 배경 |

### 테마 선택
- 부모가 시작 시 테마 선택
- 테마 변경도 부모만 가능 (특수 조합)

## 페이지 구조

```
[시작 화면]
├── 테마 선택 (6개 카드)
├── 시작하기 버튼 → 풀스크린 진입
│
[놀이 화면] — 풀스크린
├── 키 입력 감지 → 비주얼 효과 렌더링
├── 테마별 배경
├── 시스템 키 차단
└── 부모 종료 조합 → 시작 화면으로

[부모 설정] (선택)
├── 테마 변경
├── 효과 강도 조절
└── 타이머 설정 (N분 후 자동 종료)
```

## 기술 스택
- **Frontend Only**: HTML + CSS + JavaScript (또는 React + Vite)
- **서버 불필요**: 정적 사이트. API 없음.
- **애니메이션**: CSS Animation + Canvas API (파티클 효과)
- **호스팅**: Railway (정적) 또는 Cloudflare Pages (무료)
- **PWA**: manifest.json + 풀스크린 모드

## 수익 모델
- 시작 화면에 광고 (놀이 화면에는 광고 없음 — 아기가 누를 수 있으니까)
- 테마 선택 화면에 광고
- 놀이 종료 후 광고

## UI 컨셉
- 밝고 알록달록한 색상
- 둥근 모서리, 큰 버튼
- 아기가 봐도 즐거운 디자인
- 폰트: 둥글둥글한 서체

## 키 차단 구현 가이드

```javascript
// 모든 키 이벤트 캡처
document.addEventListener('keydown', (e) => {
  // 시스템 단축키 차단
  e.preventDefault();
  e.stopPropagation();

  // 비주얼 효과 트리거
  triggerEffect(e.key);
}, true);

// 우클릭 차단
document.addEventListener('contextmenu', (e) => e.preventDefault());

// beforeunload 차단
window.addEventListener('beforeunload', (e) => {
  e.preventDefault();
  e.returnValue = '';
});

// 부모 종료: Ctrl+Shift+Q 5초 유지
let exitTimer = null;
document.addEventListener('keydown', (e) => {
  if (e.ctrlKey && e.shiftKey && e.key === 'Q') {
    if (!exitTimer) {
      exitTimer = setTimeout(() => exitToHome(), 5000);
    }
  }
});
document.addEventListener('keyup', () => {
  clearTimeout(exitTimer);
  exitTimer = null;
});
```

## 파티클 효과 참고

Canvas API로 파티클 시스템:
- 키 누를 때마다 누른 위치(랜덤) 에서 파티클 생성
- 파티클: 원, 별, 하트, 이미지 등
- 중력+마찰 적용으로 자연스러운 낙하
- 테마별 파티클 색상/모양 변경

## 이미지 리소스
- 무료 이모지 또는 SVG 아이콘 사용
- 또는 이모지 자체를 렌더링 (외부 리소스 불필요)
- 이모지로 하면 이미지 로딩 없이 바로 표시
